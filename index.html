Responsive Sticky Navigation Menu Bar on Scroll using HTML CSS & JavaScript

1. What is a Sticky Navigation Menu?
A sticky navigation menu (or "sticky navbar") is a menu that stays fixed at the top of the webpage even when the user scrolls down. This allows users to access the navigation easily from any point on the page, improving the overall user experience. It remains "stuck" at the top while scrolling, hence the name "sticky."

2. Key Features of a Sticky Navbar:
Fixed positioning: The navbar stays at the top of the viewport. Scroll-triggered effects: The navbar can change styles (e.g., color, size) when the user scrolls. Responsiveness: The navbar adjusts to different screen sizes for mobile, tablet, and desktop views. Hover effects: Adds an interactive touch, highlighting links when hovered.
